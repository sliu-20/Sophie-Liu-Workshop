    The green, yellow, and red buttons are classified as "success button", "alert button", and "warning button" respectfully.

    There are also multiple ways to create buttons. The expanded button is created using 

            <button type = "expanded button">
    while the button groups are created using 
            <a class = "button">

    This is a callout. The box resizes to fit the page, so adjust your window to see how it works.

    The text always stays in the box, with roughly the same amount of space.

    A callout is created using 

    <div class="callout">

    A callout can be colored 5 different ways, all based off the color palette.

    This is a primary callout.

    Code: 

    <div class="primary callout">

    The amount of space between the text and the box can also be manipulated.

    This is a small secondary callout.

    Code: 

     <div class="secondary callout small">

    This is a large success callout.

     <div class="success callout large">
